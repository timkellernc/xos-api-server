API Reference

This is a reference guide for Harmonic XOS API Bridge.

Overview

The API provides programmatic access to certain features of the XOS encoder. All endpoints are RESTful and return JSON responses. You can communicate with multiple XOS devices by providing an IP address in the API request.

Authentication

There is currently no authentication required in this app, but this is subject to change if I feel like it. 

Current API Key: DONT_WORRY_ABOUT_IT

Endpoints

POST /api/scte

Sends a SCTE command to XOS Encoder

Parameters

Name Type Description
endpoint string XOS ESAM Endpoint (optional)
stream_id string ESAM endpoint ID
upid string Segmentation UPID

Example Response

{
    "status": "ok",
    "response": "<Response [403]>"
 }
GET /api/esam-endpoint

Get the default endpoint that Harmonic ESAM messages should be sent to.

Example Response

{
    "endpoint": "192.168.11.25:8088"
 }
